

## Улучшение Toast-уведомлений

### Текущие проблемы

1. **Нет иконок** -- toast показывает только текст, без визуальной индикации типа (success/error/warning/info)
2. **Лимит 1 toast** -- `TOAST_LIMIT = 1`, новый toast вытесняет предыдущий
3. **Огромный delay** -- `TOAST_REMOVE_DELAY = 1000000` (~16 минут), toast практически не исчезает сам
4. **Варианты success/warning/info заливают фон целиком** -- слишком агрессивно, лучше мягкие стили с цветной полоской
5. **Showcase не использует варианты** -- кнопки Warning/Info/Success вызывают toast без variant, показывая default стиль
6. **Нет progress bar** для визуализации автозакрытия

### План изменений

#### 1. `src/hooks/use-toast.ts`
- Увеличить `TOAST_LIMIT` до 3 (показывать стопку уведомлений)
- Уменьшить `TOAST_REMOVE_DELAY` до 5000 мс (5 секунд, адекватное автозакрытие)
- Добавить опциональное поле `icon` в тип `ToasterToast` (`React.ReactNode`)

#### 2. `src/components/ui/toast.tsx`
- Переработать стили вариантов: вместо полной заливки -- белый/темный фон с цветной левой полоской (`border-l-4`) и мягким оттенком фона
- Варианты:
  - `default` -- как есть
  - `destructive` -- красная полоска, светло-красный фон
  - `success` -- зеленая полоска, светло-зеленый фон
  - `warning` -- оранжевая полоска, светло-оранжевый фон  
  - `info` -- синяя полоска, светло-синий фон

#### 3. `src/components/ui/toaster.tsx`
- Рендерить `icon` рядом с title/description (если передан)
- Добавить анимированный progress bar внизу toast для визуализации оставшегося времени (5 сек CSS animation)

#### 4. `src/components/showcase/NotificationShowcase.tsx`
- Кнопки Success/Warning/Info -- передавать соответствующий `variant` и `icon`
- Добавить пример toast с `action` (кнопка "Отменить")
- Обновить описание внизу -- указать все доступные варианты

### Визуальный результат

```text
┌─────────────────────────────────┐
│ ▌ ✓  Расчёт сохранён        ×  │
│ ▌    График обновлён успешно    │
│ ▌                               │
│ ████████████░░░░ progress bar   │
└─────────────────────────────────┘
```

Тосты стекаются до 3 штук, автоматически закрываются через 5 секунд, каждый вариант визуально отличается цветной полоской и иконкой.
